Users API Home Page

API Endpoints

  • GET /api/users — Get all users
  • GET /api/users/:id — Get single user by ID
  • POST /api/users/adduser — Add a new user (send JSON body with username, email, password)
  • PUT /api/users/change/:id — Update user by ID (send JSON body with new data)
  • DELETE /api/users/delete/:id — Delete user by ID

How to Use

Use Postman, Insomnia, Curl or any REST client to test the API.

  • Set the URL to http://localhost:5000/api/users
  • Choose the correct HTTP method (GET, POST, PUT, DELETE)
  • If POST or PUT, add JSON body in request
  • Click Send to get the response
